# Content Editor Guide

## Quick Start Guide for Non-Technical Users

This guide explains how to create and manage content in the LGIHE admin panel without needing to know HTML or technical details.

---

## Creating New Content

### Step 1: Navigate to the Content Type
- Click on **Events**, **News**, or **Job Listings** in the sidebar
- Click the **"New"** or **"Create"** button

### Step 2: Fill in the Details

#### Title
- Type your title naturally (e.g., "Annual Graduation Ceremony 2026")
- The system will automatically create a web-friendly URL (slug)

#### Content Fields with Rich Text Editor

You'll see a text editor with formatting buttons at the top. Here's what each button does:

**Basic Formatting**:
- **B** - Make text bold
- **I** - Make text italic
- **U** - Underline text
- **S** - Strikethrough text

**Lists**:
- **Bullet List** - Create bullet points
- **Numbered List** - Create numbered lists

**Structure**:
- **H2** - Large heading
- **H3** - Medium heading
- **Quote** - Blockquote for highlighting text

**Other**:
- **Link** - Add clickable links
- **Code** - Format code or technical text
- **Undo/Redo** - Undo or redo your changes

### Step 3: Add Images (if applicable)
- Click the **"Choose file"** button
- Select an image from your computer
- Use the built-in editor to crop or adjust if needed

### Step 4: Choose How to Save

You'll see two buttons at the bottom:

#### "Save as Draft" Button
- Use this when you're not ready to publish
- The content will be saved but **NOT visible** to the public
- You can come back and edit it later
- No confirmation needed - just click and save

#### "Publish Now" Button
- Use this when you're ready to make the content public
- A popup will ask: **"Are you sure you want to publish?"**
- Click **"Yes, Publish"** to confirm
- The content becomes **immediately visible** on the website

## Understanding Status Colors

When viewing lists of content, you'll see colored badges showing the status:

### Events
- 🔘 **Gray (Draft)** - Not published yet
- 🟢 **Green (Published)** - Live on the website
- 🔴 **Red (Cancelled)** - Event was cancelled
- 🟠 **Orange (Archived)** - Old event, no longer shown

### News
- 🔘 **Gray (Draft)** - Not published yet
- 🟢 **Green (Published)** - Live on the website
- 🟠 **Orange (Archived)** - Old news, no longer shown

### Job Listings
- 🔘 **Gray (Draft)** - Not published yet
- 🟢 **Green (Active)** - Accepting applications
- 🟠 **Orange (Closed)** - No longer accepting applications
- 🔴 **Red (Archived)** - Old listing, removed from site

## Editing Existing Content

### Step 1: Find the Content
- Go to the content type (Events, News, or Job Listings)
- Find the item you want to edit
- Click the **pencil icon** or the item title

### Step 2: Make Your Changes
- Edit any field using the rich text editor
- The formatting buttons work the same way

### Step 3: Save Your Changes
- Click **"Save changes"** at the bottom
- If you change the status, the visibility will update accordingly

## Tips for Using the Rich Text Editor

### Creating Lists

**For Bullet Points**:
1. Click the bullet list button
2. Type your first point
3. Press Enter to create the next point
4. Press Enter twice to exit the list

**For Numbered Lists**:
1. Click the numbered list button
2. Type your first point
3. Press Enter to create the next point
4. Press Enter twice to exit the list

### Adding Links

1. Select the text you want to make clickable
2. Click the link button
3. Paste or type the URL
4. Click "Set link"

### Formatting Text

1. Select the text you want to format
2. Click the appropriate button (Bold, Italic, etc.)
3. The text will be formatted automatically

### Creating Headings

1. Place your cursor on the line you want to make a heading
2. Click H2 for a large heading or H3 for a medium heading
3. Type your heading text

## Common Questions

### Q: Do I need to know HTML?
**A:** No! The rich text editor handles all the HTML for you. Just use the formatting buttons.

### Q: What happens if I click "Publish Now" by mistake?
**A:** You can edit the content and change the status back to "Draft" to hide it from the public.

### Q: Can I preview my content before publishing?
**A:** The rich text editor shows you how your content will look as you type. What you see is what you get!

### Q: How do I add bold text?
**A:** Select the text and click the **B** button, or click **B** first and then type.

### Q: Can I undo changes?
**A:** Yes! Click the undo button (↶) in the editor, or use Ctrl+Z (Windows) or Cmd+Z (Mac).

### Q: What's the difference between Draft and Published?
**A:** 
- **Draft** = Only you can see it in the admin panel
- **Published** = Everyone can see it on the public website

### Q: How do I create a new paragraph?
**A:** Just press Enter once. Press Enter twice to add extra space between paragraphs.

## Best Practices

### Writing Content

✅ **Do**:
- Write clear, concise titles
- Use headings to organize long content
- Use bullet points for lists
- Add links to relevant pages
- Proofread before publishing

❌ **Don't**:
- Use ALL CAPS (it looks like shouting)
- Overuse bold or italic formatting
- Create very long paragraphs
- Forget to add images where appropriate

### Publishing Workflow

1. **Draft** - Write and format your content
2. **Review** - Check for errors and formatting
3. **Preview** - Make sure everything looks good
4. **Publish** - Click "Publish Now" when ready

### Images

- Use high-quality images
- Make sure images are relevant to the content
- Use the image editor to crop appropriately
- Keep file sizes reasonable (under 2MB)
